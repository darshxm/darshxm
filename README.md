## About

I work on data collection, analysis, and visualization, mainly turning raw information into something actionable as my day job. Currently deepening my knowledge of cloud-native architectures and applied AI, however I am seeing the benefit of moving towards a self managed server each day. My background spans process design, user journeys, and full-stack development, and some system admin.

If it's repetitive, I'll automate it. If it's complex, I'll find the pattern.
